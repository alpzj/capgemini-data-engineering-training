# 🧑‍💼 SQL Training Project – Employee Data Analysis

## 📌 Project Overview

This project demonstrates a comprehensive analysis of employee data using SQL.  
It focuses on applying real-world business logic to solve HR-related analytical problems such as salary management, performance evaluation, risk assessment, promotions, and workforce ranking.

The project was completed as part of a **Capgemini Training Program**, with the goal of strengthening practical SQL skills and understanding how data is used in business decision-making.

---

## 🎯 Objective

The main objectives of this project are:

- To understand how SQL is used in real-world HR analytics
- To apply conditional logic for business decision-making
- To analyze employee performance and compensation
- To evaluate risk and promotion eligibility
- To implement ranking techniques for data analysis

---

## 🧠 Key Concepts Used

This project demonstrates the following SQL concepts:

### 1. Conditional Logic (CASE Statements)
Used to implement business rules such as:
- Salary hike calculations based on experience and performance
- Bonus distribution based on department and rating
- Employee classification into performance categories
- Risk level evaluation for employees
- Tax calculation based on salary slabs

---

### 2. Nested Conditional Logic
Advanced CASE logic was used to handle complex business conditions such as:
- Promotion eligibility decisions
- Multi-level salary increment structures
- Combined evaluation of salary, experience, and performance

---

### 3. Data Analysis Techniques
Applied analytical thinking to:
- Categorize employees into different performance groups
- Identify high-value and low-performing employees
- Simulate real-world HR decision-making scenarios

---

### 4. Window Functions (Analytical Ranking)
Used advanced SQL ranking techniques to:
- Assign unique rankings to employees based on salary
- Rank employees within departments
- Analyze joining order and seniority
- Compare employees using different ranking methods (standard rank vs dense rank)

---

## 📊 Business Problems Solved

This project simulates real HR business scenarios:

- Determining salary hikes based on employee performance
- Calculating bonuses using department-specific rules
- Identifying high, mid, and low performers
- Assessing employee risk levels for HR planning
- Evaluating promotion eligibility for career growth
- Computing tax obligations based on salary structure
- Ranking employees for performance and seniority analysis
