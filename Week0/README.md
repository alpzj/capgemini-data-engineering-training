## 📌 Problem Understanding
The goal of this task is to understand how basic SQL operations can be implemented using PySpark.  
We work with a simple customer dataset and perform operations like filtering, selecting columns, and aggregation.

This helps in building a strong foundation for handling large-scale data using PySpark.

---

## 🛠️ Approach Taken
- Created a structured dataset with customer details  
- Applied basic data operations such as filtering and grouping  
- Used PySpark DataFrame transformations to process the data  
- Compared outputs to understand how PySpark works  

---

## 🔍 Operations Performed
- Retrieved all records  
- Filtered customers based on city  
- Filtered customers based on age  
- Selected specific columns  
- Grouped data by city and counted customers  

---

## 🔑 Key Transformations
- Filtering data using conditions  
- Selecting required columns  
- Grouping data for aggregation  
- Counting records in each group  

---

## 📘 Learnings
- Learned how SQL queries map to PySpark operations  
- Understood DataFrame transformations like filter, select, and groupBy  
- Gained hands-on experience with PySpark basics  
- Built foundational knowledge for big data processing  

---

## ⚠️ Challenges Faced
- Converting SQL logic into PySpark syntax  
- Understanding DataFrame operations  
- Reading PySpark output format  
- Initial setup and execution of Spark
